A simple number guessing game where the user guesses a randomly generated number between 1 and 100.

The project was initially implemented as a **Python console application** and later adapted into a **web-based version** for browser interaction.

## Features
- Random number generation
- Feedback on each guess (Too High / Too Low)
- Attempt counter

## Tech Stack
- Python (initial version)
- HTML, CSS, JavaScript (web version)
